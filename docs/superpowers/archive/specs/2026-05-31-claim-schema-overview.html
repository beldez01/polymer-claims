K

The claim object

top-level IR tuple
A claim is a typed structure: grammatical by form (syntax), interpreted by ontology + measurement (semantics), licensed by a scoped warrant (the bridge). The tuple below is the wire format every protocol stage reads and writes.
Claim = {
  schema_version: "v1.3",  pattern: {id, version},  profile: {id, version},
  subject:    SubjectRef          # polymorphic, 9 kinds + composite (v1.2)
  roles:      {predictor, outcome, confounder, mediator, collider, instrument}  # typed slots
  premises:   [Premise]            # L0 leaves + grounding
  operations: [Operation]          # the DAG: join / aggregate / estimator …
  statistics: [Statistic]          # produced values, byte-faithful
  inference:  InferenceRule        # AND/OR/NOT/CMP, Kleene 3-valued
  conclusion: Proposition          # L1: content + inferential neighborhood
  strength:   StrengthVector       # 6-axis Pareto (not a scalar)
  status:     Status               # CONJECTURED…LICENSED/REJECTED
  provenance: Provenance           # produced_by + licensed_by + frame + generated_by…
  relations:  [Edge]               # L3: undermine/undercut/rebut/reclassify/reinterpret
}
L0

Leaf — the empirical anchor

empirical pole
A sum-typed empirical anchor with a frame and dual provenance. The single highest-leverage primitive: it is what turns a grammar of statistics into a grammar of science — qualitative, mechanistic, and existence claims finally get a real home instead of being costumed as fake numbers.

Leaf (sum type)

new v1.3Sensitivity. Sum-typed so non-quantitative findings aren't lost.
Leaf =
  | Quantity(value, unit, uncertainty, measurement_basis)
  | Categorical(ontology_term, assay)              # ChromHMM state, allele identity
  | Existence(observed | not_detected, detection_limit) # genuine vs untested absence
  | Proposition(data, warrant, backing, qualifier, rebuttal)  # Toulmin 1958

measurement_basis = Fundamental(rep_theorem) | Derived(formula, sources)
                  | Conventional | Ordinal | Nominal
Only Quantity · Fundamental asserts a UCUM unit + meaningfulness class (Luce/Narens). Derived statistics (β, partial-ρ, fold-change) carry their generating formula, never a false unit (Roberts).

L2

Claim — pattern-typed DAG + licensing

the bridge
The pattern is the structural discriminator; the DAG computes statistics; the licensing bridge turns evidence into an entitlement; strength is a vector.

Pattern registry (axis-derived)

new v1.3Replaces the 45/47 tautology with a real disjoint object.
Pattern = {estimand, adjustment_role, null_model, scale,
           invariance_group, intended_applications, excluded_applications[]}
# MECE holds over the statistical-form AXES only — never over biology.
# merge: partial_correlation_with_control + model_delta_over_baseline → adjusted_effect
adjusted_effectenrichment_vs_nullgroup_contrast_effect_size simple_correlationranking_by_propertycount_or_threshold evidence_synthesis_pooled_effectmechanism / causal ★
Open registry → reports a coverage metric, never closure. The mechanism/causal pattern sits outside the associational catalog (conclusion = causal edge/path); associational claims attach via evidence_for edges.

Strength — 6-axis Pareto vector

new v1.3Kills the hidden scalar / Arrovian impossibility.
StrengthVector = ⟨ magnitude, uncertainty, evidence_against_null,
                  severity, world_contact, explanatory_virtue ⟩
# AND = componentwise meet · OR = join · many claims are strength-incomparable
# LICENSED ⇔ dominates a declared threshold vector on EVERY axis (no trade-off)
world_contact = the legacy EvidenceClass (M>R>D>S>K>H>L) — it survives as one input axis, not the whole of strength. Total ranking only via a logged, named-dictatorship policy.
L3

Corpus — value-based defeat graph

composition · social
Claims compose into a Value-Based Argumentation Framework (Bench-Capon 2003). Status = the set of grounded extensions; blame under contradiction is surfaced, not laundered.

Defeat edge taxonomy

new v1.3VAF → inherit PTIME-grounded tractability theorems.
edgeattacks
underminea premise / leafnew
undercutthe inferential link (warrant)new
rebutthe conclusion directlynew
reclassifythe analytic/synthetic force-type assignment itselfnew
reinterpretmeaning moved, statistics unchanged (interpretive drift)protocol
evidence_forsupports (reverse edge; e.g. assoc → mechanism)new
Duhem–Quine blame surfaced: contradiction emits the set of minimal blame-assignments → corpus status = intersection (robustly-IN) / union (possibly-IN) / difference → PENDING duhem_underdetermined.
L4

Revision — AGM/TMS over the corpus

temporal change
Corpus change is a first-class belief-revision algebra, not a static recompute.
ops: expand | revise | contract     # Alchourrón–Gärdenfors–Makinson 1985
entrenchment: ordering keyed to evidence_class + severity   # which claim yields on contradiction
justifications: produced_by / licensed_by DAG  # = a JTMS (Doyle 1979); cone-scoped propagation

# status recompute under a FIXED graph = PTIME-monotone (bounded defeat in-degree).
# graph EDITS = non-monotonic AGM ops. (corrects the false "monotone fixpoint" claim)
Entrenchment is made cone-size-invariant (a claim isn't stronger just because cheap children were built on it) — kills the Matthew effect; a single decisive high-cost result can overturn a tower of cheap correlations.
